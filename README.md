## 💡 Key Capabilities

The software handles several types of data extraction to keep your workflow simple. 

### Metadata Extraction
Every post contains hidden information. The scraper collects the date, author name, and engagement counts. It organizes these details into a simple text file for you to review later.

### Comment Harvesting
Reading through hundreds of replies takes time. This tool gathers every comment into a single list. You can search this list for specific keywords or trends.

### Video Transcription and Analysis
If you provide a link to a video, the program performs two tasks. First, it generates a full text transcript of the audio. Second, the AI engine reviews the visual content. It provides a summary of what happens in the clip, which saves you from watching long recordings.

### Multi-Platform Support
You do not need different tools for different sites. This scraper works the same way for Instagram, TikTok, Twitter, and YouTube. The interface stays consistent so you spend less time learning how to use the buttons.

## ❓ Frequently Asked Questions

Read these answers to common questions to help solve issues quickly.

### Where does the data go?
The program creates an Exports folder in your Documents directory. You can move these files to any location on your computer.

### Does this work with private posts?
The tool only accesses public content. It cannot scrape posts that require a private account login or restricted permissions.

### Why do I see a command window?
The program runs a background task to process the data. This window shows the progress of the current scrape. Do not close this window until the program finishes its work.

### Can I change the file format?
Future versions will include settings for different output formats like CSV or PDF. For now, the program uses standard text files which you can open in any word processor.

## 🛠️ Troubleshooting

If the program stops working, follow these suggestions to fix it:

1. Check your internet connection. A slow connection can cause the scraper to timeout.
2. Verify the link is valid. Paste the link into your web browser to make sure the post still exists.
3. Clear your temporary files. Sometimes an old file interferes with the new data capture.
4. Restart your computer. This clears the memory and forces a clean start for the software.

If these steps do not help, ensure your Windows system features all the latest updates. Most issues arise from outdated system files.
